## Commands

```bash
# Install Eas CLI
npm install -g eas-cli
eas login
eas build:configure

# Install Expo CLI
npm install -g expo-cli

# Development Start
expo start
npm run start

# Development Start Web
expo web
npm run web

# Production Build
eas build --platform android
```
